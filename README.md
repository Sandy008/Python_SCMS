# Python Scientific Computing (MSc) - Course Repository

## Repository Purpose

This repository contains all the Python codes, assignments, and implementations of topics discussed in the **MSc Scientific Computing** class. It serves as a comprehensive collection of computational methods, algorithms, and practical implementations used in scientific research and engineering applications.

## Course Overview

Scientific Computing is an interdisciplinary field that uses mathematical models, numerical analysis, and computer science to solve complex scientific and engineering problems. This course focuses on Python as the primary programming language for implementing computational solutions.

## Topics Covered

### 1. **Fundamentals of Scientific Computing**
- Introduction to Python for scientific applications
- NumPy for numerical computing
- SciPy for scientific algorithms
- Matplotlib for data visualization
- Jupyter notebooks for interactive computing

### 2. **Numerical Methods**
- **Root Finding**: Bisection, Newton-Raphson, Secant methods
- **Linear Algebra**: Matrix operations, eigenvalue problems, LU decomposition
- **Interpolation**: Polynomial, spline, and Lagrange interpolation
- **Numerical Integration**: Trapezoidal rule, Simpson's rule, Gaussian quadrature
- **Differential Equations**: Euler's method, Runge-Kutta methods

### 3. **Data Analysis and Statistics**
- Statistical analysis with Python
- Data manipulation with Pandas
- Probability distributions and random sampling
- Hypothesis testing and regression analysis
- Monte Carlo methods

### 4. **Optimization**
- Linear programming
- Nonlinear optimization
- Genetic algorithms
- Gradient descent methods
- Constrained and unconstrained optimization

### 5. **Computational Physics and Engineering**
- Finite difference methods
- Finite element analysis basics
- Heat transfer simulations
- Wave equation solutions
- Particle simulations

### 6. **Machine Learning in Scientific Computing**
- Supervised and unsupervised learning
- Neural networks for scientific applications  
- Feature selection and dimensionality reduction
- Cross-validation and model selection

### 7. **High-Performance Computing**
- Vectorization techniques
- Parallel computing with multiprocessing
- Memory optimization
- Profiling and performance analysis

### 8. **Specialized Applications**
- Image processing and computer vision
- Signal processing
- Bioinformatics applications
- Climate and environmental modeling
- Financial modeling

## Learning Objectives

By the end of this course, students will be able to:

1. **Apply computational thinking** to solve complex scientific problems
2. **Implement numerical algorithms** from scratch using Python
3. **Analyze and visualize data** effectively using scientific Python libraries
4. **Optimize code performance** for large-scale computations
5. **Design and implement simulations** for physical and engineering systems
6. **Apply machine learning techniques** to scientific data
7. **Write well-documented, reproducible code** following best practices

## Key Technologies and Libraries

- **Core Python**: Object-oriented programming, data structures
- **NumPy**: N-dimensional arrays, mathematical functions
- **SciPy**: Scientific algorithms and statistical functions
- **Matplotlib/Seaborn**: Data visualization and plotting
- **Pandas**: Data manipulation and analysis
- **Scikit-learn**: Machine learning algorithms
- **SymPy**: Symbolic mathematics
- **Jupyter**: Interactive computing environment

## Getting Started

### Prerequisites
- Basic programming knowledge
- Fundamental mathematics (calculus, linear algebra, statistics)
- Understanding of scientific/engineering principles

### Installation
```bash
# Create a virtual environment
python -m venv scms_env
source scms_env/bin/activate  # On Windows: scms_env\Scripts\activate

# Install required packages
pip install numpy scipy matplotlib pandas scikit-learn jupyter sympy seaborn
```
